# Harvest-Helper
Веб-приложение, призванное упростить труд работников сельского хозяйства путем предоставления прогноза погоды с последующим анализом данных и предоставления советов по сельскохозяйственным работам.

Задача сервиса: должна быть возможность выбрать город, для выбранного города отображается погода. Сервис должен предоставлять подробную информацию, такую как влажность, давление. На основании прогноза погоды должны предоставляться советы пользователю. Если пользователь хочет увидеть советы, например, для посадки малины, то он выбирает в приложении эту культуру, затем сервис анализирует прогноз погоды с апреля по май, находит ближайший период, когда впоследствии не предвидится заморозков, и сообщает пользователю эту дату с пометкой, что это благоприятный период для начала сельскохозяйственных работ.
